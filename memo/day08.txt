반복문
	특정 조건동안 코드를 반복 실행한다.


for문
	for(변수 in 반복가능대상){
		반복 실행할 코드
	
	}
	* 반복 가능대상의 데이터의 개수가 반복 횟수이다

while문
	while(조건식){
		반복 실행할 코드
	}
	* 조건식이 참인 동안 코드를 반복 실행한다
	* 조건식이 거짓인 경우가 존재해야 무한루프(반복)에 빠지지 않는다

repeat문
	
	repeat{
		반복 실행할 코드
	}

	* 코드를 무한 반복 실행한다	
	* if문과  break를 통해 특정 조건을 만족하면 반복을 종료시킨다

break 와 next
	반복문(루프) 안에서 사용한다.
	break : 반복을 종료하고 중괄호 영역을 벗어난다.
	next: 다음 반복으로 넘어간다(건너띄기)

apply() 함수
	apply(data,margin,FUN)
		데이터의 행과 또는 열방향에 함수를 적용한다
		margin = 1(행방향),2(열방향),3(면방향)
		* 함수를 작성할 때 ()소괄호를 작성하지 않는다
	lapply(data,FUN)
		데이터에 함수를 적용한다
		데이터를 리스트로 변환한 후에 함수를 적용한다.
	sapply(data,FUN)
		데이터에 함수를 적용한다
		데이터를 리스트로 변환한 후에 함수를 적용한다

질적 데이터와 양적 데이터
	질적 데이터(범주형)
		명목형 : 범주들 간의 순서가 의미 없는것(혈액형,성별,출생지)
		순서형 : 범주들 간의 순서가 의미 있는것(순위,학력,선호도)

	양적데이터(수치형)
		연속형 : 셀 수 없는 것(시간, 키, 온도, 무게, 길이)
		이산형 : 셀 수 있는 것(수강생의 수, 과목의 점수, 교통사고의 수)

기술통계량
	1.평균(mean)
	2.합계(sum)
	3.중위수(중앙값)(median) : 여러개의 값 중에 중앙에 위치한 값
	4.최빈수(mode) : 빈도가 가장 많은 수
	5.사분위수(quartile) : 값들을 크기의 순서대로 정렬하고 4등분하여 구하는 4개의 값
	6.분산(variance) : 값들이 퍼져있는 정도
	7.표준편차(standard diviarion) : 분산을 제곱급한 것
	8.최소값(minimum) : 값들 중에 가장 작은 값
	9.최대값(maximum) : 값들 중에 가장 큰 값
	10.범위(range) : 최소값 ~ 최대값
	11.첨도(kurtosis) : 분포의 비대칭 정도
	12.왜도(skewness) : 분포의 뾰족함 정도


사용자 정의 함수(function)
	R에 내장되어 있는 함수가 아닌 사용자가 원하는
	기능을 위해 직접 만든 함수

	-> 특정 기능을 위한 코드의 묶음
	
	함수명<- function(매개변수1,매개변수2,,,){
		실행할 코드
		...
		return(리턴값)
	}
	
	매개변수(parameter) : 외부에서 함수로 전달하는 값을 저장하는 변수
	인수(argument) : 외부에서 함수로 전달되는 값
	리턴값 : 함수에서 외부로 반환하는 값	

	 ex) f(x) = x + 1 -> 매개변수 : x
	      f(2) -> 인수 : 2
	** 인수의 값이 매개변수에 저장되어 함수의 중괄호 안에서 사용된다

	print_hello <- function() {
		print('hello')
	}

매개변수와 리턴값
	매개변수가 있다면 외부에서 함수로 값을 전달할 수 있다
	매개변수가 없다면 외부에서 함수로 값을 전할할 수 없다.
	
	함수를 호출 할때는 인수와 매개변수의 개수가 동일해야 한다	
	
	리턴값이 없다면 함수 내부의 값을 외부로 전달하지 못한다
	리턴값이 있다며 함수 내부의 값을 외부로 전달할 수 있고,
	함수 호출 코드 자체를 하나의 값으로 볼 수 있다
	
	리턴값은 1개만 가능하며, 여러개의 값을 리턴하고 싶다면 자료구조에 담아 리턴한다.
