빅데이터(Big Data)
	대량의 복잡한 데이터들의 모음
	빅데이터의 특징 : 3V(5V, 7V, 9V..)

	- Volume(규모)	: 거대한 규모의 데이터
	- Variety(종류)	: 다양한 데이터의 형태와 범위
	- Velocity(속도)	: 빠른 데이터 처리 및 분석

	-> 거대한 규모의 다양한 데이터를 빠른 수집과 처리를 하면서 
	나타나는 현상

데이터의 유형
	1. 정형 데이터 : 구조화된 형태의 데이터(규칙이 존재)
		-> DBMS(데이터베이스), Excel, Csv 등...
	2. 비정형 데이터 : 구조화되지 않은 형태의 데이터(규칙이 존재 X)
		-> 텍스트 문서, 이미지, 비디오, 오디오, 소셜 미디어 게시물..
	3. 반정형 데이터 : 구조화는 되어 있지 않지만, 규칙이 존재하는 데이터
		-> XML, JSON 및 웹 스크랩된 데이터 등..

빅데이터 처리 과정
	데이터 수집 > 데이터 정제(전처리) > 데이터 분석
	> 데이터 시각화 > 데이터 활용(머신러닝, 딥러닝 등...)

R의 특징
	1. 통계학자가 만든 언어로 다양한 통계 정보를 산출이 가능하다
	2. 그래픽 처리 기능이 뛰어나다
	3. 다양한 패키지 제공(다른 사람들이 만들어 놓은 기능)
	4. 한글에 취약. 윈도우 계정이름, 폴더 이름이 한글로 되어 있으면
	문제가 발생할 수 있다
	5. 사용자 간의 커뮤니티가 발달되어 있다
	
R과 파이썬
	R, 파이썬(Python) -> 무료
	통계 분석 소프트웨어(프로그램) -> SAS, SPSS -> 유료
	
	* R: 프로그래밍 언어이나 파이썬에 비해 데이터분석(통계분석)
	에 특화되어 있다
	* 파이썬(Python): 프로그래밍 언어로 데이터분석 뿐만 아니라
	어플리케이션 개발 등 범용으로 사용 가능하다

	* R 공식 홈페이지
	https://www.r-project.org/

	다양한 작업을 사용할수 있도록 도와준다

R 스튜디오 인터페이스
	1. Script : 스크립트 파일을 보여준다
	2. Enviroment : 현재 사용중인 데이터를 보여준다
	3. Console : 코드 실행 결과를 보여준다
	4. Files : 작업공간 안의 여러개의 파일들을 보여준다

스크립트 파일
	여러개의 코드를 저장해놓은 파일
	* 콘솔창에서 코드를 입력하여 사용하면 코드를 해석하거나
	재사용하기 힘들기 때문에 사용할 코드들을 하나의 스크립트 파일에
	저장하여 사용한다

설정
	- 패널 위치 변경하기 tools > global options > Pane Layout에서 설정
	- 명령어가 길어졌을 시 자동으로 줄바꿈 tools > global options > code > soft wrap도 체크하기 (물리적인 공간이 부족하면 자동 줄바꿈)
	- 글씨 크기 및 글씨체 설정 Tools > Global Options > Appearance > 글씨 크기, 글꼴, 테마
	- 기본 workspace 설정 : Tools > Global Options.. > General > Default Working directory...
	- 인코딩 설정 : Tools > Global Options ... > Code >Saving > Default Text Encoding > UTF-8 변경

주석(comment)
	주석 처리가 된 코드는 컴퓨터가 실행하지 않는다
	
	1. 해당 코드를 보는 다른 사람들에게 설명글을 작성하기 위해서 사용
	2. 코드를 잠시 실행대상에서 제외하고 싶을때 사용
	* 단축키 : ctrl + shift + c

단축키
	- 새로운 스크립트 파일 만들기 : ctrl + shift + n
	- 한줄의 코드 실행 : ctrl + enter(실행하고 다음줄로 이동) / alt + enter(실행하고 다음줄로 이동 x)
	- 여러줄의 코드 실행 : 실행할 코드 드래그 -> ctrl + enter
	- 화면 확대 및 축소 : ctrl + shift + ‘+’ : 확대 ctrl + ‘-’ : 축소
	- 스크립트의 모든 코드 실행 방법 : ctrl + shift + r
	- 한줄의 코드 삭제 : ctrl + d
	- 코드 위아래로 이동 : alt + 위아래 방향키
