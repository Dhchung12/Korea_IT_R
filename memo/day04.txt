day04. 벡터, 매트릭스

복습
	벡터 생성하기
		1. c() 함수
			-> 일반적으로 가장 많이 사용하는 방법
		2. seq() 함수
			-> 규칙성이 있는 숫자 벡터를 생성하는 방법
		3. 콜론연산자(:)
			-> 규칙성이 있는 숫자 벡터를 생성하는 방법
		4. rep() 함수
			-> 기존 벡터를 활용하여 새로운 벡터를 
			생성하는 방법

벡터에서 요소 가져오기
	1. 벡터에서 하나의 요소 가져오기
		벡터명[인덱스번호]

	2. 벡터에서 여러개의 요소 가져오기
		벡터명[여러개의 인덱스번호가 담긴 벡터]
		ex) 벡터명[c(1, 3)]
		     벡터명[1:4]
		     벡터명[seq(1, 10, 2)]...		
		** 벡터에서 대괄호 안에 ,(쉼표)를 사용하면
		차원이 다르기때문에 에러가 발생한다

	3. 벡터에서 특정 요소를 제외하고 가져오기
		벡터의 요소에 부여되는 인덱스 번호는 항상 1부터
		시작하고, 1씩 증가한다 => 음수 인덱스가 존재하지 않음

		벡터명[-인덱스번호]
		작성한 인덱스 번호를 제외한 나머지 요소를 가져온다

		** 벡터명[-(제외할 여러개의 인덱스번호가 담긴 벡터)]

	4. 조건을 통해 특정 요소를 가져오기
		1. 논리벡터를 생성한다
		2. 생성한 논리벡터를 가져오려고 하는 벡터의
		인덱스 자리에 작성한다

		* 벡터에서 논리값을 인덱스로 사용하면
		전체 요소에 논리값이 순차적으로 부여되고
		부여된 논리값이 TRUE 인 요소만 가져온다

		* num_vec <- 1:10
		-> 짝수만 가져오기

		(1) 논리벡터 생성
			num_vec %% 2 == 0
		(2) 논리벡터를 인덱스 자리에서 작성
			num_vec[num_vec %% 2 == 0]
	
벡터의 정보
	1. str() : 벡터의 자료형, 인덱스번호, 요소를 보여준다
	2. length() : 벡터의 길이(요소의 개수)를 보여준다
	3. names() : 벡터의 열 이름을 가져온다
		names(벡터명) <- c("이름1", "이름2", "이름3"...)
		* 벡터의 열 이름은 중복되면 안되고, 가능한 한글은
		사용하지 않는다
		* 벡터의 열 이름을 지정하면 인덱스 번호 대신
		열 이름으로 요소를 가져올수 있다


벡터의 사용
	1. 벡터에 값 추가하기(삽입)
		벡터명[새로운 인덱스번호] <- 새로운값
		벡터명[새로운 열 이름] <- 새로운 값

	2. 벡터의 값 삭제하기
		벡터명 <- 벡터명[-삭제할 요소의 인덱스번호]

	3. 벡터의 값 수정하기
		벡터명[수정할 요소의 인덱스번호] <- 새로운값

매트릭스(matrix)
	행렬.
	같은 자료형의 여러개의 데이터를 2차원 구조로 저장한다
	행 : 가로줄, row
	열 : 세로줄, column

매트릭스 생성
	matrix(벡터)
	벡터를 기반으로 행렬을 생성한다
	ex) 20칸 짜리 벡터를 4행 5열의 매트릭스로 생성한다

	* data : 매트릭스로 생성할 여러개의 값들(벡터로 전달한다)
	* nrow : 행의 개수
	* ncol : 열의 개수
	* byrow : 데이터를 채우는 방향(기본설정이 열방향)

매트릭스에서 요소 가져오기

	매트릭스명[행 인덱스 번호, 열 인덱스 번호]
	** 매트릭스의 요소는 행과 열 인덱스 번호 2개를 가지고 있다

	1. 하나의 행 요소를 가져오기 => 가로줄로 데이터 가져오기
		매트릭스명[행 인덱스번호, ]

	2. 하나의 열 요소를 가져오기 => 세로줄로 데이터 가져오기
		매트릭스명[, 열 인덱스번호]

	3. 하나의 요소 가져오기
		매트릭스명[행 인덱스번호, 열 인덱스번호]
