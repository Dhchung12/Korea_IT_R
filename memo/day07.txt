day07. 요인, 제어문

요인(factor)
	범주형 데이터를 표현하는 벡터
	범주형 데이터 : 성별(여, 남), 혈액형(A, AB, B, O)
		선호도(매우 만족, 만족, 보통, 불만족, 매우 불만족)
	
	지정된 범위의 값만 저장할 수 있도록 제한한 벡터

R의 자료구조
	1. 스칼라 : 길이가 1인 벡터 -> 하나의 값만 저장한다
	2. 벡터 : 같은 자료형의 여러개의 값을 저장한다 -> 1차원
	3. 매트릭스(행렬) : 같은 자료형의 여러개의 값을 저장한다
			-> 2차원(행과 열을 이용하여 데이터를
			저장한다)
	4. 배열 : 같은 자료형의 여러개의 값을 저장한다
			-> N차원(행과 열과 면을 이용하여
			데이터를 저장한다 -> 여러개의 행렬)
	5. 데이터프레임 : 각 열마다 다른 자료형의 값을 저장할 수 
			있는 행렬
	6. 리스트 : 기본 벡터와 다르게 각 자료형을 다르게 데이터를
		저장할 수 있다-> 여러개의 자료구조를 저장한다
		[벡터, 배열], [벡터, 데이터프레임]
	* 요인 : 지정한 범주 내의 값만 사용할 수 있도록 제한하는 벡터

제어문
	코드의 실행을 제어한다
	1. 조건문
		특정 조건을 만족하면 코드를 실행한다
		(if문, switch 함수)
	2. 반복문
		특정 코드를 반복 실행한다
		(for문, while문, repeat)

조건식
	코드의 실행결과가 TRUE 또는 FALSE 둘 중 하나로 표현되는 코드

if문
	조건식의 결과가 TRUE(참)이면 중괄호 안의 코드를 실행한다

	if(조건식) { # 블록의 시작
		조건식의 값이 TRUE이면 중괄호 안의 코드를 실행한다
		
	} # 블록의 끝

	if(조건식) {
		조건식의 값이 TRUE이면 실행할 코드
	}else{
		조건식의 값이 FALSE이면 실행할 코드
	}

	if(조건식1) {
		조건식1의 값이 TRUE이면 실행할 코드
	}else if(조건식2) {
		조건식1의 값이 FALSE이고,
		조건식2의 값이 TRUE이면 실행할 코드
	}else{
		조건식1과 조건식2가 FALSE이면 실행할 코드
	}

ifelse() 함수
	ifelse(조건식, 참일경우 실행할 코드, 거짓일 경우 실행할코드)
	** 조건식에 해당하는 논리값이 여러개여도 순차적으로 사용가능

switch() 함수
	비교대상과 조건값의 일치 여부에 따라 결과값을 다르게 사용한다
	switch(비교대상, 조건값1 = 결과값1, 조건값2 = 결과값2....)

for문
	반복 가능 대상에서 요소를 1개씩 꺼내 변수에 담으며
	코드를 반복 실행한다

	for(변수 in 반복횟수 -> 반복 가능 대상) {
		반복 실행할 코드 -> print("hello")
	}

	ex) print("hello") -> 100번 실행하고 싶다
		1) 해당 코드를 100번 작성해서 1번 실행한다
		2) 해당 코드를 작성하고 100번 실행한다
		3) 반복문을 사용한다
