day09. 사용자 정의 함수, 파일입출력, 패키지

사용자 정의 함수
	함수명 <- fuction(매개변수1, 매개변수2...) {
		실행할 코드
		...
		return(리턴값)
	}

	* 외부에서 함수에 값을 전달하기 위해 매개변수를 사용
	* 함수에서 외부에 값을 전달하기 위해 리턴값을 사용
	* 함수에서 생성한 변수는 지역변수 이다
	* 함수 밖에서 생성한 변수는 전역변수 이다

매개변수와 리턴값
	함수에 값을 전달할때는 매개변수를 사용
	함수에서 외부에 값을 전달할때는 리턴값을 사용
	-> 리턴값이 있다면 함수 호출 코드를 값으로 볼 수 있다

	리턴값은 1개만 가능하며 여러개를 리턴하고 싶다면
	자료구조 안에 담아 자료구조를 리턴한다

가변 매개 인자(인수)
	매개변수의 개수를 지정하지 않고, 전달하는 인수의
	개수만큼 사용 할 수 있다

	함수명 <- function(...) {
		실행할 코드
	}

디폴트 매개변수(기본값 매개변수)
	매개변수에 값을 미리 지정해서 사용한다
	만약 인수가 전달되면 전달된 값으로, 전달되지 않으면	
	지정한 값으로 사용한다

	함수명 <- function(매개변수명 = 기본값) {
		실행할 코드
	}

파일 입출력(I/O)
	파일 입력(Input) -> 읽기
	파일 출력(Output) -> 쓰기

	R 프로그램 		외부파일
		<------(입력, 읽기)---
		--------(출력, 쓰기)-->


절대경로와 상대경로
	1. 절대경로 : 파일이나 디렉토리의 전체 경로(드라이브부터 시작)
	2. 상대경로 : 현재 작업 디렉토리를 기준으로 파일이나 디렉토리의 위치

	* / : root
	* . : 현재위치
	* .. : 상위경로

인코딩과 디코딩
	인코딩 : 문자 -> 이진 숫자로 변환(암호화)
	디코딩 : 이진 숫자 -> 문자로 변환(복호화)

	파일을 생성할때 사용한 규칙과 파일을 읽을때 사용한 규칙이
	동일해야 데이터를 읽을수 있다
	ASCII , UTF-8, EUC-KR(윈도우즈) 등...
